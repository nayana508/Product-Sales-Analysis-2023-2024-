# Product-Sales-Analysis-2023-2024-
This project presents an interactive Power BI dashboard built to analyze product sales data for the years 2023 and 2024. The goal is to transform raw sales data into meaningful insights that support data-driven business decisions.
The dataset contains 200,000 records and 14 features, including order details, customer information, product categories, revenue, and profit. Using Power BI, the data is cleaned, modeled, and visualized to uncover key patterns and trends.
🔍 Key Features
Analysis of overall sales performance
Identification of best-selling and underperforming products
Regional sales distribution insights
Monthly sales trend analysis
Profitability analysis by category and sub-category
📈 Dashboard Highlights
KPIs: Total Revenue, Total Profit, Quantity Sold
Visualizations:
Bar Chart (Top Sub-Categories)
Tree Map (Sales by Region)
Donut Chart (Profit by Category)
Line Chart (Monthly Sales Trends)
Pie Chart (Revenue by Category)
Filters (Slicers): Category, Year, State
📊 Key Insights
Electronics generate the highest revenue
East region leads in overall sales
Home & Furniture is the most profitable category
Sales peak during November and December
🛠 Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Data Visualization Techniques
📌 Conclusion
This project demonstrates how business intelligence tools can convert raw data into actionable insights, helping organizations improve performance, optimize strategies, and identify growth opportunities.
